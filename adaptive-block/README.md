# Портфолио сотрудника - Петров Василий

Веб-страница портфолио сотрудника с адаптивным дизайном.

## 📁 Структура проекта

```
portfolio/
├── index.html    # Главная HTML-страница
├── styles.css    # Стили и адаптивный дизайн
└── script.js     # JavaScript логика
```

## ✨ Реализованные функции

### 1. Блоки с личной информацией
- ✅ Имя и должность в шапке
- ✅ Основная информация (ФИО, страна, город, контакты)
- ✅ Образование
- ✅ Проекты с описанием и технологиями
- ✅ Отзывы коллег

### 2. Система наград и достижений
- ✅ **Структура данных наград** с полями:
  - `name` - название награды (с эмодзи)
  - `description` - описание, показываемое при наведении
  - `backgroundUrl` - градиент для фона награды
  
- ✅ **Карусель наград** с навигацией
- ✅ **Адаптивная сетка**: 
  - Десктоп: 6 наград на экране (2 ряда)
  - Планшет: 6 наград
  - Мобильный: 6 наград (3x2)
  
- ✅ **Индикация страниц** (точки внизу)
- ✅ **Тултипы** с описанием при наведении
- ✅ **Анимации** при взаимодействии

### 3. Дополнительные фичи
- ✅ Радар-диаграмма навыков (SVG)
- ✅ Счетчик баллов достижений (164 балла)
- ✅ Градиентные фоны
- ✅ Плавные анимации и переходы

## 📱 Адаптивность

Страница полностью адаптивна для:
- 💻 Десктоп (1200px+)
- 📱 Планшеты (768px - 1024px)
- 📱 Смартфоны (до 768px)

### Брейкпоинты:
- **1024px** - переход на планшетную версию
- **768px** - переход на мобильную версию (одна колонка)
- **480px** - оптимизация для маленьких экранов

## 🎯 Структура данных наград

```javascript
const achievements = [
    {
        name: '🏆 Первый проект',
        description: 'Успешно завершен первый проект в компании',
        backgroundUrl: 'linear-gradient(135deg, #667eea 0%, #764ba2 100%)'
    },
    // ... еще 11 наград
];
```

Всего в системе **12 наград**, которые автоматически распределяются по страницам.

## 💡 Решение для множества наград

Когда наград много и они не помещаются на экран, реализовано:

1. **Пагинация** - награды разбиты на страницы по 6 штук
2. **Кнопки навигации** - стрелки влево/вправо для перехода между страницами
3. **Точечная индикация** - показывает текущую страницу
4. **Автоматическая адаптация** - количество наград на странице меняется в зависимости от размера экрана
5. **Smooth transitions** - плавная анимация при переключении

## 📝 Как добавить новую награду

Откройте `script.js` и добавьте объект в массив `achievements`:

```javascript
{
    name: '🎯 Название',
    description: 'Описание награды',
    backgroundUrl: 'linear-gradient(135deg, #цвет1, #цвет2)'
}
```

Награда автоматически появится в карусели!

## 🔧 Настройка

Все данные легко настраиваются через переменные в `script.js`:
- `achievements` - массив наград
- `skills` - навыки для диаграммы
- `achievementsPerPage` - количество наград на странице
